# 漫画サムネイルメーカー

YouTube動画や販売ページ用のサムネイル画像を簡単に作成できるChrome拡張機能です。背景画像と漫画の表紙画像を組み合わせて、ドロップシャドウ付きのプロフェッショナルなサムネイルを作成できます。

![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-green?logo=googlechrome)
![Manifest V3](https://img.shields.io/badge/Manifest-V3-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 機能

- **サイドパネルUI** - ポップアップではなく広いサイドパネルで作業
- **10種類の背景画像** - 6種類のプリセット + 4スロットのユーザー登録枠
- **カスタム背景登録** - 独自の背景画像を登録して使用可能
- **自動リサイズ** - アスペクト比を維持して最適なサイズに調整
- **ドロップシャドウ** - 濃いめの影で浮遊感を演出
- **PNG保存** - 高品質なPNG形式でダウンロード

## 📦 インストール方法

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/Tener2000/manga-thumbnail-maker.git
   ```

2. Chromeで `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」をオンにする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

## 🎨 使い方

### 1. 背景画像を選択
プリセットグリッドから背景画像を選択します。
- 番号1-6: デフォルト背景（グラデーション）
- 番号7-10: ユーザー登録枠

### 2. カスタム背景を登録（オプション）
「背景画像を登録」セクションで登録番号(1-10)を指定し、独自の背景画像を登録できます。

### 3. 表紙画像をアップロード
「表紙画像をアップロード」から漫画の表紙画像を選択します。

### 4. プレビュー確認
背景の上に表紙画像がセンター配置され、ドロップシャドウが適用された状態をプレビューできます。

### 5. 画像を保存
「💾 PNG画像として保存」ボタンをクリックすると、`thumbnail_[タイムスタンプ].png` という名前でダウンロードされます。

## 📁 ファイル構成

```
漫画サムネイルメーカー/
├── manifest.json          # 拡張機能の設定
├── sidepanel.html         # サイドパネルUI
├── sidepanel.css          # スタイルシート
├── sidepanel.js           # メインロジック
├── background.js          # バックグラウンドスクリプト
├── backgrounds/           # 背景画像プリセット
│   ├── bg1.png           # パープル〜ピンク
│   ├── bg2.png           # ブルー〜シアン
│   ├── bg3.png           # オレンジ〜イエロー
│   ├── bg4.png           # レッド〜マゼンタ
│   ├── bg5.png           # グリーン〜ティール
│   └── bg6.png           # ネイビー〜ライトブルー
└── icons/                 # 拡張機能アイコン
    ├── icon16.png
    ├── icon48.png
    ├── icon64.png
    └── icon128.png
```

## 🔧 技術仕様

| 項目 | 値 |
|------|-----|
| キャンバスサイズ | 1280 x 720px (16:9) |
| 余白 | 上下左右各5% |
| ドロップシャドウ | 35pxぼかし、18pxオフセット |
| 出力形式 | PNG |

## 🛠️ 使用技術

- **Chrome Extension Manifest V3**
- **Side Panel API**
- **Canvas API** - 画像合成
- **Chrome Storage API** - ユーザー設定保存
